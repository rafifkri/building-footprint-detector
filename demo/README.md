# Demo Video

Place your demo video (demo_video.gif) here after creating it.

To create a demo GIF:

1. Screen record the Streamlit app or inference process
2. Convert to GIF using tools like:
   - ffmpeg: `ffmpeg -i video.mp4 -vf "fps=10,scale=800:-1" demo_video.gif`
   - Online tools: ezgif.com, giphy.com
